Многозадачное расширение для браузеров.

Возможности:

  • Улучшение производительности
    При сильной нагрузке браузера, расширение может освободить память и ослабить нагрузку на процессор путём заморозки неактивных вкладок.

  • Быстрая закрузка файлов
    Расширение добавляет кнопку быстрого меню, которая в одно нажатие скачивает файл, не раздражая пользователя всплывающими окнами.
    Работает на изображениях, видео, аудио файлах, HTML страницах и ссылках.

  • Моментальный отклик
    Даже при нагрузке процессора на 90%, расширение способно мгновенно загрузиться, заморозить вкладки и ускорить работу браузера.
    Всё из-за минимального количества зависимостей.

  • Конвертация выбранных вкладоки в закладки
    Если вкладки вам в данный момент не нужны, но вы хотите воспользоваться ими позже, расширение может сохранить их в закладки и освободить рабочее пространство, всего за пару нажатий.

  • Назначение горячих клавиш

    3D Software Renderer

    Онлайн Демо
    GitHub

    Традиционно отрисовкой трёхмерной гафики, занимаются видеокарты. Их главным преимуществом является высокая многопоточность.

    Однкако аппаратной отрисовке есть альтернатива: программный ренденринг.

    Для улучшения своего понимания трёхмерной графики, я написал демо.

    Особенности и свойства программной отрисовки:

  • Легко портировать
    Написанно приложение на языке Rust, который имеет замечательную поддержку WebAssembly / Win / Linux / MacOS / Android и позволяет писать безопасный код, который не уступает по скорости C/C++.
    Для работы нужен лишь процессор. Что упрощает портирование кода на разные платформы.
    Включая Web браузеры!

  • Скорость
    Приложенние было скомпилированно в WebAssembly, что позволяет получить стабильно высокую частоту кадров даже на смартфоне, без использования SIMD инструкций.

  • Легковестность
    Вся логика, геометрия и математические функции встроены в один бинарный файл, который хорошо сжимается.

  • Точность
    Программой производится перспективная коррекция текстур. Поддерживается 32 бита на пиксель (включая альфа канал).
    Используя палитру можно ещё сильнее улулчшить производительность.

    • Сделано используя: Rust, WebAssembly, Sass, TypeScript, tiny_lib
