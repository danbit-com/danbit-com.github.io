Многозадачное расширение для браузеров.

Возможности:

  • Улучшение производительности
    При сильной нагрузке браузера, расширение может освободить память и ослабить нагрузку на процессор путём заморозки неактивных вкладок.

  • Быстрая закрузка файлов
    Расширение добавляет кнопку быстрого меню, которая в одно нажатие скачивает файл, не раздражая пользователя всплывающими окнами.
    Работает на изображениях, видео, аудио файлах, HTML страницах и ссылках.

  • Моментальный отклик
    Даже при нагрузке процессора на 90%, расширение способно мгновенно загрузиться, заморозить вкладки и ускорить работу браузера.
    Всё из-за минимального количества зависимостей.

  • Конвертация выбранных вкладоки в закладки
    Если вкладки вам в данный момент не нужны, но вы хотите воспользоваться ими позже, расширение может сохранить их в закладки и освободить рабочее пространство, всего за пару нажатий.

  • Назначение горячих клавиш

    • Технологии: Sass, TypeScript, Chrome API
